Deep Learning Fundamentals

Layered representations powering modern AI.

HG

Harshad Gore

Published Nov 12, 2025

Overview

Deep learning uses multiple stacked layers to transform raw inputs into increasingly abstract features. Each layer learns patterns that compose into higher-level representations.

Key Building Blocks

  • Linear / Dense Layers: Affine transform + activation.
  • Non-linear Activations: ReLU, GELU introduce capacity.
  • Loss Functions: Guide optimization (CrossEntropy, MSE).
  • Optimizers: Update weights (SGD, Adam).

Forward Pass Pseudocode

import torch
import torch.nn as nn

class SimpleMLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(32, 64), nn.ReLU(),
            nn.Linear(64, 64), nn.ReLU(),
            nn.Linear(64, 10)
        )
    def forward(self, x):
        return self.net(x)

Why Depth Works

Hierarchical Features

Early layers learn edges or token segments; deeper layers learn objects or semantic relations.
