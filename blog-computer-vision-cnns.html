How machines learn to see.

Convolutional Neural Networks

CNNs apply learned filters across spatial dimensions to extract hierarchical visual features.

Core Components

  • Convolution: Local receptive fields.
  • Pooling: Downsample activations.
  • BatchNorm: Stabilize training.
  • Fully Connected: Final decision layers.

Minimal PyTorch Example

import torch
import torch.nn as nn

class TinyCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(3, 16, 3, padding=1), nn.ReLU(),
            nn.MaxPool2d(2),
            nn.Conv2d(16, 32, 3, padding=1), nn.ReLU(),
            nn.MaxPool2d(2)
        )
        self.fc = nn.Linear(32*8*8, 10)
    def forward(self, x):
        x = self.conv(x)
        x = x.view(x.size(0), -1)
        return self.fc(x)
